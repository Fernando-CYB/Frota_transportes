# 🚚 Sistema de Gestão de Veículos - C#

Aplicação Console desenvolvida para praticar **Programação Orientada a Objetos em C#**, simulando o gerenciamento de uma frota com cálculo de consumo de combustível, pedágio e custo de manutenção.

---

## 📌 Funcionalidades

* Cadastro de veículos da frota
* Suporte a diferentes tipos de veículos:

  * Carro
  * Caminhão
* Simulação de viagem com consumo de combustível
* Cálculo automático de pedágio
* Cálculo de custo de revisão
* Tratamento de erros (combustível insuficiente ou valores inválidos)

---

## 🛠️ Tecnologias Utilizadas

* C#
* .NET Console Application

---

## 🧠 Conceitos Aplicados

* Programação Orientada a Objetos (POO)
* Classes e objetos
* Herança
* Polimorfismo
* Classes abstratas
* Interfaces
* Encapsulamento (`private set` / `protected set`)
* Tratamento de exceções (`try/catch`)
* Manipulação de `decimal`
